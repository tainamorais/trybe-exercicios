Principal dependência para rodar um projeto em TypeScript
npm install -D typescript@4.4.4

Instalar express
npm i express

Para rodar projeto TypeScript e cria tsconfig.json
npx tsc --init

(mudar target para es6, pois é compatível com node 16 alpine deste projeto, rootDir e outDir)

Para transpilar o index.ts se tudo estiver correto. No package.json tem que ter o build configurado como tsc.
npm run build

Para rodar (tem que ter o start configurado no package.json)
npm start

npm i -D ts-node-dev
npm i -D ts-node

Para instalar os types do express
npm i -D @types/express

npm i -D @types/node

Connection pool
npm install mysql2

npm install dotenv
