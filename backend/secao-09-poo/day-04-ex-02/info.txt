01. Startar projeto:
npm init -y

02. Instalar typescript e node js
npm i -D typescript ts-node

03. Iniciar com tsc para ter arquivo de configuração tsconfig
npx tsc --init

04. Instalar nodemon
npm i -D nodemon

05. Acertar package.json para dev com nodemon

06. Instala dependências
npm install

07. Cria um arquivo index.ts, coloca um console.log e testa para ver a saída
npm run dev

Outros 2 inicializadores de typescript além do nodemon (ts node dev E tsx):
npm i -D ts-node-dev tsx


RODAR COM:
npx ts-node src
